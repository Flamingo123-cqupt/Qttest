# Qt 5.12.12 无人机频谱瀑布图识别检测模块

这是一个 Qt Widgets + qmake 示例工程，用于演示“基于频谱数据瀑布图的无人机信号识别检测模块”。

## 功能

- 生成模拟频谱数据。
- 显示实时频谱图。
- 显示实时瀑布图。
- 支持环境底噪、无人机遥控窄带链路、无人机图传宽带链路、无人机跳频链路四种场景。
- 基于峰值功率、占用带宽、活跃频点数量、跳频次数做简单识别。
- 支持检测门限调节、暂停/继续、清空历史事件。
- 频谱图显示当前帧功率曲线、峰值标记和频率坐标；瀑布图显示历史频谱随时间变化。

## 编译

使用 Qt 5.12.12 的 qmake：

```bash
qmake DroneSpectrumDetector.pro
make
```

Windows MinGW 环境可使用：

```bat
qmake DroneSpectrumDetector.pro
mingw32-make
```

## 说明

当前识别算法是工程演示版规则模型，真实项目中可将 `SpectrumAnalyzer` 替换成更完整的特征库、协议特征、机器学习模型或外部检测 SDK。
